# Tech Innovators Club Website

A responsive college club website for **Tech Innovators Club | KIET**. The website includes Home, About, Events, and Join pages with improved styling, mobile navigation, event filtering, form validation, and scroll animations.

## Files Included

```text
college_club_updated/
├── index.html
├── about.html
├── events.html
├── join.html
├── style.css
├── script.js
├── README.md
└── 7528.jpg_wh1200.jpg
```

> Keep the image file name exactly as `7528.jpg_wh1200.jpg`, or update the image path in all HTML files.

## Pages

### Home Page
Contains the club introduction, hero section, call-to-action buttons, highlights, and statistics.

### About Page
Explains the purpose, mission, and vision of the club.

### Events Page
Displays club events using responsive cards. JavaScript allows users to filter events by category.

### Join Page
Contains a membership form with JavaScript validation.

## Improvements Made

- Added responsive design for mobile, tablet, and desktop screens.
- Added sticky navigation bar.
- Added mobile hamburger menu using JavaScript.
- Fixed image paths by using relative paths.
- Improved UI with cards, shadows, gradients, spacing, and hover effects.
- Added event category filter on the Events page.
- Added form validation on the Join page.
- Added scroll reveal animations.
- Added back-to-top button.
- Added better semantic HTML and accessibility labels.

## How to Run

1. Download or copy all project files into one folder.
2. Make sure `index.html`, `style.css`, `script.js`, and the image file are in the same folder.
3. Open `index.html` in any web browser.

No server or external library is required.
